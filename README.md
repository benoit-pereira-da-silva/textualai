# Raffut-ai

**Raffut-ai** provides streaming building blocks (processors) for LLM providers **and** a small, provider-agnostic **CLI chat tool** built on top of the [`textual`](https://github.com/benoit-pereira-da-silva/textual) pipeline abstractions.
